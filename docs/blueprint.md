# **App Name**: LET's Prep

## Core Features:

- Exam Simulation: Simulate the actual Licensure Examination for Teachers (LET) testing environment.
- Question Bank: Access a comprehensive question bank covering various LET subject areas.
- Adaptive Testing: The app will learn the areas of weakness for the user, and adjust future tests to emphasize the weaker area
- Performance Analysis: Provide detailed performance analysis and identify strengths and weaknesses using the 'tool'.
- Timer and Pacing: Implement a timer to help users practice effective pacing during the exam.
- Scoring and Feedback: Calculate scores and provide instant feedback after each practice test.

## Style Guidelines:

- Primary color: Light blue (#A7D9ED), reminiscent of academic settings and focus.
- Background color: Very pale blue (#F2F9FC), for a calm and neutral background.
- Accent color: Muted green (#90EE90) for positive reinforcement and correct answers.
- Body and headline font: 'PT Sans', a modern sans-serif offering both a contemporary look and some warmth, for readability.
- Use clear and simple icons for navigation and features.
- Maintain a clean and organized layout for easy navigation and a stress-free experience.
- Use subtle animations for feedback and progress indication.
